# eBayセラークエスト 🗡️

eBayセラー業務の日々の作業を **RPG風にこなす達成トラッカー**（iOS向けPWA）。

## 特徴
- 🔍 新規出品リサーチ ／ 💰 最安値チェック ／ 📦 在庫補充 の3クエストを毎日こなす
- タスク完了・作業件数で **XP獲得 → レベルアップ**、ヒーローが進化（見習い→熟練→ベテラン→マスター）
- 🔥 連続達成日数（ストリーク）
- 📅 カレンダーで過去の作業量をヒートマップ表示、日ごとの内訳も確認
- データは**端末内（localStorage）だけに保存**。サーバーには一切送信されません
- オフライン対応（Service Worker）。ホーム画面に追加してアプリのように使えます

## iPhoneでの使い方
Safariで公開URLを開き、共有ボタン → **「ホーム画面に追加」** を選ぶだけ。

## 開発
静的ファイルのみ。ローカルで動かす場合:
```
python3 -m http.server 8123
# http://localhost:8123 を開く
```
アイコンPNGは `node make-icons.js` で再生成できます（依存なし）。
